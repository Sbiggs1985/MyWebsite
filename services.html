Technologies

Info

HTML

HTML (HyperText Markup Language) is a markup language used to define the structure of web pages. It uses a system of tags to organize content into elements such as text, images, forms, and multimedia, providing the foundation for web design and interaction.

CSS

CSS (Cascading Style Sheets) is a style sheet language used to control the visual appearance of web pages. It allows you to apply styles like colors, fonts, layouts, and spacing to HTML elements, making websites more attractive and user-friendly.

JavaScript

JavaScript is a versatile scripting language that allows developers to create dynamic and interactive elements on websites. It can manipulate HTML and CSS, handle events, and communicate with servers, enabling features like interactive forms, pop-ups, and real-time content updates.

Python

Python is a versatile language offering simplicity, readability, and extensive libraries for web development, data analysis, automation, and machine learning.

MySQL

MySQL is an open-source relational database management system that uses SQL (Structured Query Language) to store, retrieve, and manage data efficiently. It's widely used in web applications for handling large-scale databases.

Bootsrap

Bootstrap is a front-end framework that enables rapid development of responsive, mobile-first websites with pre-built components.
